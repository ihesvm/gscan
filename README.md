# Gscan [Go Port Scanner]



### This program allows you to scan network ports in IP range, but the result is saved in a json file.

---
*how to use*

***The first step is to compile the program***
```bash
$ go build .
```
***then***
```bash
$ gscan tcp example.com 1024 -> ip range
```
